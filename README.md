<h2>🛠️ Installation Steps:</h2>

<p>1. Install</p>

```
pnpm install
```


<p>2. Build</p>

```
pnpm run build
```


<p>3. Run local app</p>

```
pnpm run dev
```


<p>4. Preview production</p>

```
pnpm run preview
```
